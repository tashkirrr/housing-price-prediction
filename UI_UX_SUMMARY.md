## 🎭 Animation System

### **6 Core Animations**
1. **fadeInUp** - Elements enter from below (0.6s)
2. **fadeIn** - Simple opacity transition (0.5s)
3. **slideInLeft** - Content slides from left (0.6s)
4. **slideInRight** - Content slides from right (0.6s)
5. **scaleIn** - Elements pop in (0.5s)
6. **pulse** - Subtle attention grabber

### **Micro-Interactions**
- 🌊 **Button Ripple**: Click any button
- 📦 **Card Lift**: Hover over cards
- ✨ **Border Reveal**: Country cards gradient top
- 💫 **Focus Glow**: Form inputs with accent ring
- 🎯 **Scale Active**: Buttons press down (0.98)

```

💻 Desktop: 1024px+
```
